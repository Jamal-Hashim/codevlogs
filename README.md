# 💻 Code Vlogs Website

A modern and responsive **Code Vlogs Website** built using **HTML, CSS, and JavaScript**.  
This platform is designed to share programming knowledge, tutorials, notes, courses, and curated playlists in a clean and user-friendly interface.

The website provides a structured learning experience for beginners and intermediate developers, making it easy to explore coding resources and improve skills.

---

## 🌟 Features

- 🏠 **Home Page (Index)** with featured content and highlights  
- 📝 **Notes Section** for study materials and programming concepts  
- 🎓 **Courses Page** with structured learning content  
- ▶️ **Playlists Section** for video-based learning  
- 📖 **About Page** describing the platform  
- 📞 **Contact Page** for user queries and feedback  
- 🎨 Clean and modern UI design  
- ⚡ Smooth animations and hover effects  
- 📱 Fully responsive design (Mobile + Desktop)  
- 🧭 Easy navigation menu  

---

## 🛠️ Technologies Used

- **HTML5** – Website structure  
- **CSS3** – Styling and layout  
- **JavaScript** – Interactivity and dynamic behavior  
- **Font Awesome** – Icons  
- **Responsive Web Design**
